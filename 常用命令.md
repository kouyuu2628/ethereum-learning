### Linux常用命令
返回上级目录
```
cd ..
```
查看当前路径
```
pwd
```
##### 文件操作
-  查看文件
```
cat [文件名]
```
- 创建文件
```
touch [文件名]
```
- 修改文件名(a.txt -> b.txt),如果存在b.txt,则覆盖
```
mv a.txt b.txt
```
- 删除文件
```
rm [文件名]
```
- 删除文件(强制删除)
```
rm -f [文件名]
```

##### 文件夹操作
- 向下递归，不管有多少级目录，一并删除，会有提示
```
rm -r [目录]
```
- 直接强行删除，不作任何提示的意思
```
rm -rf [目录]
```
